# 💰 BudgetBuddy - Personal Finance Management System

BudgetBuddy is a full-stack web application that helps users efficiently manage their personal finances by tracking income and expenses, organizing categories, managing bank accounts, and generating detailed reports.

The application provides a secure authentication system, interactive dashboard, dynamic charts, and exportable reports, making personal finance management simple and effective.

---

## 📌 Features

### 🔐 User Authentication

- User Registration
- User Login
- Secure Session Management
- Only authenticated users can access the application
- Every user's data is stored separately

---

### 💵 Transaction Management

Users can add both **Income** and **Expense** transactions.

Transaction Form includes:

- Transaction Type (Income / Expense)
- Category
- Date
- Payment Mode (Online / Offline)
- Amount
- Select Bank
- Description
- Payment Details
- Receipt Upload
- Dynamic Submit Button
  - 🟢 Add Income
  - 🔴 Add Expense

All transaction data is stored dynamically in MySQL.

---

### 📂 Category Management

Users can create unlimited custom categories.

Fields:

- Category Type
  - Income
  - Expense
- Category Name

Categories can also be:

- Edited
- Deleted
- Viewed

---

### 🏦 Bank Management

Users can store multiple bank accounts.

Bank Details include:

- Account Holder Name
- Account Number
- Bank Name
- Branch
- IFSC Code

Supports:

- Add Bank
- Delete Bank
- View Bank List

---

### 👤 Account Management

Store multiple personal or business accounts.

Fields:

- Account Name
- Account Number
- Email

Supports:

- Add Account
- Delete Account
- View Account List

---

### 📊 Interactive Dashboard

The dashboard provides graphical financial analysis using Chart.js.

Includes:

- Today's Expense
- Monthly Expense
- Monthly Income
- Yearly Expense
- Yearly Income

Dashboard updates automatically according to the logged-in user's transactions.

---

### 📄 View Records

Income Records

- View all income transactions
- Receipt preview
- Edit transaction
- Delete transaction

Expense Records

- View all expense transactions
- Receipt preview
- Edit transaction
- Delete transaction

### 📈 Reports

Income & Expense Report
